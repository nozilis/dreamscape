# Dreamscape

Сервис для тех, кто хочет собрать свои мечты в одном месте
и видеть как они исполняются.

## Почему Dreamscape

Todo-листы слишком утилитарны для жизненных целей.
Заметки теряются. Таблицы не вдохновляют.

Dreamscape — это пространство где мечты живут: 
добавляешь желание, указываешь категорию, отмечаешь 
когда исполнилось. Сайт создаёт атмосферу а не просто 
хранит данные.

Идея появилась из личной потребности — не нашёл 
удобного решения, сделал своё.
